# Hylo 🚧  
*A custom programming language by Brewlyn Surge*

## Project Status: In Development 🔥

Hylo is an experimental, custom-designed programming language currently **under active development**.  
As the language evolves, syntax, features, and internal architecture may change significantly.

Because this project is still in its early stages:

- The language is **not yet stable**
- Documentation is incomplete and may change frequently
- The compiler/interpreter may not work as expected

## Stay Updated

Development is ongoing — check back for updates, new features, and documentation improvements.
